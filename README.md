# Criando um Container de uma Aplicação WEB utilizando o docker-composer
  Desafio do curso da DIO "Formação Docker Fundamentals"


1. Definição dos arquivos de uma Aplicação WEB
2. Criação de um arquivo YML com as definições de um servidor Apache (httpd) e especificando o local onde os arquivos da aplicação estão localizados


## Dicas
1. Para colocar a aplicação no ar, execute o comando **docker-compose up -d** no terminal
2. Se você precisa parar os serviços, execute o comando **docker-compose down** no terminal
